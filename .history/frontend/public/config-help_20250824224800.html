Konfigurasi Server Chatbot Sekolah

Solusi untuk masalah GEMINI_API_KEY yang belum terkonfigurasi

Masalah yang Ditemui

Setelah mendeploy aplikasi chatbot sekolah ke Vercel, muncul pesan error:

Konfigurasi server belum lengkap (GEMINI_API_KEY).

Ini menunjukkan bahwa environment variable GEMINI_API_KEY belum dikonfigurasi pada server Vercel.

Solusi 1: Menambahkan Environment Variable di Vercel

  1. Login ke dashboard Vercel di https://vercel.com
  2. Pilih project chatbot sekolah Anda
  3. Pergi ke Settings > Environment Variables
  4. Klik "Add New" untuk menambahkan variable baru
  5. Isi Name dengan GEMINI_API_KEY
  6. Isi Value dengan API key Gemini Anda
  7. Pilih Environment: Production (dan Development jika diperlukan)
  8. Klik Save
  9. Redeploy aplikasi Anda
Pelajari lebih lanjut

Solusi 2: Konfigurasi File Environment Lokal

Untuk pengembangan lokal, buat file .env.local di folder frontend dengan konten:

GEMINI_API_KEY=your_actual_gemini_api_key_here
Penting: Pastikan file .env.local ditambahkan ke .gitignore agar tidak tercommit ke repository publik yang dapat membocorkan API key Anda.

Contoh isi file .gitignore:

# Environment variables
.env
.env.local
.env.development.local
.env.test.local
.env.production.local

Cara Mendapatkan Gemini API Key

Jika Anda belum memiliki API key untuk Google Gemini, ikuti langkah berikut:

  1. Kunjungi Google MakerSuite
  2. Login dengan akun Google Anda
  3. Klik "Create API Key"
  4. Salin API key yang dihasilkan
  5. Gunakan API key tersebut dalam konfigurasi
Dapatkan API Key

Testing Konfigurasi

Setelah mengkonfigurasi environment variable, lakukan testing dengan:

# Deploy ulang aplikasi
npx vercel --prod

# Atau untuk deployment preview
npx vercel

Pastikan tidak ada error terkait GEMINI_API_KEY pada proses deployment.
